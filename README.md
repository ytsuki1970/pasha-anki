# すごい暗記帳

写真でおぼえる、自分だけの暗記帳アプリ。スマホ・PC のブラウザだけで動きます（インストール不要・外部サーバー不要・無料）。

公開URL（GitHub Pages）: https://ytsuki1970.github.io/sugoi-ankicho/

## 特長
- 複数の暗記帳（デッキ）を作成・色分け
- 写真つきの暗記カード
- 3つの学習モード：フラッシュカード／4択クイズ／タイピング
- おぼえた管理（Leitner方式）・連続学習日数・正解で紙吹雪
- 📷 写真からまとめてカード作成（**端末内**で文字読み取り＝Tesseract.js）
- データは各端末のブラウザ内（IndexedDB）に保存。**サーバーには一切送信しません**
- オフライン対応（サービスワーカー）。スマホは「ホーム画面に追加」でアプリのように使えます

## 使い方
このページ（URL）を開くだけ。最初からサンプルの暗記帳が入っています。
スマホでは共有メニューから「ホーム画面に追加」してください。

## 技術
- 単一の `index.html`（フレームワークなし・バニラ JS）
- 外部 API / 有料サービスなし
- 文字読み取りライブラリ（Tesseract.js）のみ CDN から任意・遅延読み込み

> 「撮るだけで自動で問題作成」する AI 機能は今後の拡張候補です。
